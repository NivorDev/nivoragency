# 🚀 Nivor Agency - Landing Page

Landing page moderna y responsive desarrollada para una agencia digital ficticia, enfocada en ofrecer una experiencia visual atractiva, alto rendimiento y una estructura optimizada para SEO.

## ✨ Características

* 🎨 Diseño moderno y minimalista
* 📱 Responsive para móviles, tablets y escritorio
* 🌙 Modo oscuro y modo claro
* 🍔 Menú hamburguesa para dispositivos móviles
* ⚡ Animaciones suaves al hacer scroll
* 🖼️ Hero con efectos interactivos
* 📨 Formulario de contacto funcional
* 💻 Sección de tecnologías utilizadas
* 🔍 SEO básico implementado
* 🚀 Optimización de imágenes y rendimiento

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Flexbox
* CSS Grid
* Intersection Observer API
* LocalStorage
* Devicon Icons

## 🚀 Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/NivorDev/Nivor-Agency.git
```

2. Entra en la carpeta del proyecto:

```bash
cd NivorAgency
```

3. Abre `index.html` en tu navegador o utiliza una extensión como **Live Server** en Visual Studio Code.
